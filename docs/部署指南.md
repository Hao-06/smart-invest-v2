> **目标**:把 `app/dashboard_v2.py` 部署到 Streamlit Cloud,得到一个评委直接点开就能看的公网链接
> **前置**:GitHub 仓库已上线 → https://github.com/Hao-06/smart-invest-v2 ✅
> **耗时**:约 5 分钟操作 + 3-5 分钟等部署
> **费用**:**免费**(Streamlit Community Cloud 公开 repo 不要钱)

## 步骤 1 · 登录 Streamlit Cloud

打开 → https://share.streamlit.io/

点 **Continue with GitHub** → 用 **Hao-06** 账号登录 → 授权 Streamlit Cloud 读取你的 repo 列表。

> 首次登录会要求授权 OAuth,**只勾选「Public repos」**就够了,不需要给私有 repo 权限。

---

## 步骤 2 · 新建 App

进入 dashboard 后右上角点 **「Create app」** → 选 **「Yup, I have an app」**(因为代码已经在 GitHub 上了)。

填表单:

| 字段 | 值 |
|---|---|
| Repository | `Hao-06/smart-invest-v2` |
| Branch | `main` |
| Main file path | `app/dashboard_v2.py` |
| App URL | 可自定义,推荐:`smart-invest-v2`(完整地址变成 `smart-invest-v2.streamlit.app`)|

---

## 步骤 3 · Advanced settings(可选,默认体验不需要)

**默认体验完全不需要配置任何 secrets** —— 看板的 Tab 1 / Tab 2 / Tab 3(offline 模式)全部从 `runs/v2_*.json` 加载,零网络依赖。

**仅当你想在 Tab 3 启用「LLM 推理 (R1)」复选框** 才需要配置 secret:
1. 点 Advanced settings → Python version 选 **3.13**
2. Secrets 框里填:
   ```toml
   DEEPSEEK_API_KEY = "sk-你的真实 key"
   ```
3. 这个 secret 只存在 Streamlit Cloud 后台,**不会进 GitHub**

---

## 步骤 4 · Deploy

点底部蓝色 **Deploy!** 按钮。

会跳到部署日志页 —— 大约:
- **30 秒** clone repo
- **2-3 分钟** pip install requirements.txt(akshare + streamlit + plotly + pandas 等)
- **20 秒** 启动 Streamlit

成功后自动跳转到看板,**网址就是你的上线链接**:
```
https://smart-invest-v2.streamlit.app
```
(如果选了自定义 URL;否则是 `https://hao-06-smart-invest-v2-app-dashboard-v2-xxxxx.streamlit.app`)

## 常见问题

### Q1. pip install 失败
99% 是 `akshare` 装不上(它依赖一些可选包)。Streamlit Cloud 给的报错日志能定位。
最常见的是 `lxml` 版本冲突 —— 加一行到 `requirements.txt`:
```
lxml>=4.9
```
然后 push,Streamlit Cloud 自动重新部署。

### Q2. 启动后 Tab 1 报错"找不到 runs/v2_continuous_2024_2026.json"
检查 `runs/v2_continuous_2024_2026.json` 在 GitHub repo 上确实存在(打开
https://github.com/Hao-06/smart-invest-v2/tree/main/runs 看)。如果不在,运行:
```bash
git add runs/v2_continuous_2024_2026.json runs/v2_region_*.json runs/v2_recent.json
git commit -m "data: 补齐 V2 报告"
git push
```

### Q3. Tab 3「生成推荐」转圈不出结果
99% 是 akshare 拉行情时被云端 IP 限制。Streamlit Cloud 的服务器在境外,akshare 默认源
(东方财富 / 腾讯财经)对境外 IP 偶尔不稳定。**这是已知限制,不影响 Tab 1 / Tab 2 的展示**。
如果实在要修,可以加代理(需要付费方案)。**演示主路径用 Tab 1/2 即可**。

### Q4. 部署后想停掉
Streamlit Cloud dashboard 上点 app 旁边的 ⋯ → Delete。
